Quick hack to enable ligatures in Xcode.

## Requirements

- OS X El Capitan
- Xcode 7.0

## Installation

* open project file,
* build.

There's `Copy Files` - `Build Phase`, which automatically copies plugin to `Plug-ins` directory.
